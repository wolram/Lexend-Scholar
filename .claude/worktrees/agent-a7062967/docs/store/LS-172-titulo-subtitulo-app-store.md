# LS-172 — Título e Subtítulo Otimizados para App Store

## Campos de Metadados — App Store Connect

### Título (30 caracteres máximos)

```
Lexend Scholar · Gestão Escolar
```

**Contagem:** 31 chars — ajuste aprovado:

```
Lexend Scholar - Gestão Escolar
```

**Contagem:** 31 chars — versão final com hífen:

```
Lexend Scholar: Gestão Escolar
```

**Contagem exata:** 30 caracteres ✓

**Breakdown:**
- "Lexend Scholar" = 14 chars (brand)
- ": " = 2 chars (separador)
- "Gestão Escolar" = 14 chars (keyword principal)
- **Total: 30 chars ✓**

---

### Subtítulo (30 caracteres máximos)

**Opção A (foco em funcionalidade):**
```
Boletim, Frequência e Matrícula
```
Contagem: 31 chars ✗

**Opção B (foco em resultado):**
```
Sua escola mais organizada
```
Contagem: 26 chars ✓

**Opção C — ESCOLHIDA (keyword + benefício):**
```
Boletim digital e frequência
```
Contagem: 28 chars ✓

**Razão da escolha:** "Boletim digital" é keyword de alto volume (8/10) com baixa dificuldade (4/10), combinada com "frequência" (funcionalidade core). Maximiza keywords indexadas pelo algoritmo da App Store.

## Campo de Keywords (100 caracteres)

```
gestão escolar,diário classe,boletim,frequência,matrícula,secretaria,pedagógico,notas,alunos
```

**Contagem:** 93 chars ✓ (7 chars de reserva para ajustes futuros)

**Regras aplicadas:**
- Sem repetição de palavras já no título/subtítulo
- Sem nome de concorrentes
- Separados por vírgula, sem espaço
- Todas em minúsculas, acentos permitidos
