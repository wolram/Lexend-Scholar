# LS-169 — Pesquisa de Palavras-chave App Store (Categoria Education)

## Resumo Executivo

Pesquisa de keywords para o Lexend Scholar na App Store brasileira, categoria Education.
Objetivo: identificar as 10 melhores palavras-chave em português com maior volume de busca
e menor dificuldade de ranqueamento.

## Metodologia

- Ferramentas utilizadas: AppFollow, Sensor Tower e App Store Search Ads Recommendations
- Mercado: Brasil (pt-BR)
- Categoria: Education (6017)
- Data de referência: Abril 2026
