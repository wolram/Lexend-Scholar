## Overview

Lexend Scholar is a multi-surface school management system built for Brazilian educational institutions — covering student lifecycle, academic workflows, scheduling, and AI-assisted operations.

## Platform Surfaces

```
website/    # Institutional & marketing (HTML/CSS)
webapp/     # Web application (React · TypeScript)
iosapp/     # iOS native app (Swift · SwiftUI)
android/    # Android app
```

## Key Features

- 🎓 Student enrollment and lifecycle management
- 📅 Scheduling and calendar system
- 🤖 AI-assisted academic workflows
- 📱 Native iOS app (SwiftUI)
- 🤖 Android support
- 🌐 Institutional website with pricing and blog

## Tech Stack

| Layer | Technology |
|---|---|
| Web App | React · TypeScript · Node.js |
| iOS | Swift · SwiftUI · XcodeGen |
| Android | Android native |
